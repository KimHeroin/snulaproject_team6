Considering different device specifications, we prepared different versions of this game.
Original concept is based on 3d interaction, also it uses URP(Universal Render Pipeline) from unity, which can cause frame/graphic errors to some devices.
So, we prepared a web-built javascript game for universal visitors. It can also be used as a quick preview of the gameplay.

To play 3d demo version, click here to download the game.
*This 3d version will be updated steadily until the page is shut down.

여러가지 게임 플레이 환경을 고려하여, 우리 팀은 여러가지 게임 버전을 준비했습니다.
최초 게임 구상은 3d 맵 상에서의 인터랙션을 기반으로 하며, UNITY의 URP 개발환경을 이용하였습니다. 이로 인해 저사양 환경은 프레임이나 그래픽 에러가 발생할 수 있습니다.
그래서, 저희는 모든 접속자들을 위해 웹 기반 자바스크립트 게임을 준비했습니다. 이는 오리지널 게임의 빠른 프리로 쓰일 수 있습니다.

3D 데모 버전 플레이를 위해서 여기를 클릭하시오.
*이 게임은 페이지 배포가 종료될때까지 상시 업데이트 됩니다.
